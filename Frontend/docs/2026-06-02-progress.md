```

```

Runtime:

```kotlin
org.tensorflow.lite.Interpreter
```

The app records WAV at:

```text
16kHz mono 16-bit PCM
```

The converted model expects:

```text
input: [1, 192000] float32
duration: 12 seconds
output: [1, 5] logits
labels: Happy, Sad, Angry, Fearful, Neutral
```

Short recordings are zero-padded. Long recordings are truncated to 12 seconds.

### Current-Turn Prompt Injection

Voice emotion is not stored as RAG. It is injected only into the current user prompt because emotion can change every utterance.

Injected block:

```text
[음성 감정 분석]
녹음된 사용자 음성의 주요 감정은 ...
전체 분포: ...

[응답 조정 지침]
...
```

The original audio attachment is still sent to Gemma/LiteRT-LM when direct attachment mode is enabled.

## Prompt Architecture

Prompt inputs are separated into three layers.

### Fixed Policy Instruction

Defined in:

```text
Frontend/app/src/main/java/com/example/counseling/ChatPromptUtils.kt
```

Constant:

```kotlin
fixedCounselingPolicy
```

This is not user-editable.

Includes:

- Korean counseling assistant behavior.
- Crisis handling.
- Dynamic analysis interpretation rules.
- WHO 5-17 youth physical activity guidance.
- Mindfulness and stabilization guidance.

### User Custom Response Instruction

Defined as:

```kotlin
defaultUserCustomInstruction
```

Users can edit this in the prompt/settings UI. It should only control:

- Tone.
- Answer length.
- Formatting preference.
- Response style.

Legacy saved full system prompts are normalized so they do not overwrite fixed policy.

### Dynamic Context

Injected into the latest user message only when relevant:

- Memory/RAG search.
- Health Connect summary.
- Phenotype/life-pattern summary.
- Gallery analysis summary.
- Current voice emotion analysis.

Current user text always has priority over stale analysis.

## WHO Youth Activity Guidance

WHO 5-17 physical activity guidance was placed in fixed policy, not repeated in every Health prompt.

Policy includes:

- Average at least 60 minutes/day of moderate-to-vigorous physical activity.
- Vigorous aerobic activity at least 3 days/week.
- Muscle and bone strengthening activity at least 3 days/week.
- Reduce sedentary behavior.
- Reduce recreational screen time.
- Start small and gradually increase frequency, intensity, and duration.
- Treat activity patterns as auxiliary counseling context, not diagnosis.
