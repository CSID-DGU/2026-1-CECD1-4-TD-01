# Counseling: 온디바이스 상담 보조 앱

Counseling은 사용자의 대화, 갤러리 이미지, 건강 데이터, 통화 기록, 앱 사용 패턴을 바탕으로 현재 상태를 정리하고 온디바이스 Gemma가 상담 흐름을 보조하는 모바일 앱입니다.

이 앱은 진단이나 치료를 대신하지 않습니다. 사용자가 자신의 감정과 상황을 안전하게 정리하고, 다음 행동을 스스로 선택할 수 있도록 돕는 상담 보조 도구입니다.

## 핵심 아이디어

- **온디바이스 상담**: 민감한 상담 맥락을 기기 내부에서 처리하는 방향을 지향합니다.
- **맥락 기반 응답**: 사용자의 말뿐 아니라 갤러리, 건강, 생활 패턴을 보조 맥락으로 참고합니다.
- **안전 우선**: 자해, 폭력, 학대, 긴급 위험 신호가 있으면 일반 조언보다 안전 확인을 우선합니다.
- **사용자 통제**: 권한 허용, 권한 줄이기, 데이터 제외, 기간 설정, 모델 선택을 사용자가 직접 조절합니다.

## 전체 사용 흐름

1. 사용자는 상담 탭에서 현재 고민을 입력합니다.
2. 앱은 사용자가 허용한 데이터만 바탕으로 보조 맥락을 구성합니다.
3. 갤러리 분석은 최근 이미지에서 생활 맥락과 안전 신호를 요약합니다.
4. 건강 탭은 활동량, 심박, 수면 등 생활 리듬을 정리합니다.
5. 생활 패턴 탭은 통화 기록과 앱 사용 정보로 사회적 연결, 고립 신호, 디지털 과사용, 야간 리듬을 요약합니다.
6. 상담 엔진은 이 정보를 진단 근거가 아니라 대화 보조 자료로만 사용합니다.
7. 사용자는 설정에서 모델, 응답 맥락, 데이터 제외 항목, 표시 크기, 테마를 조정할 수 있습니다.

## 화면 구성

### 1. 상담

상담 화면은 사용자가 고민을 입력하고 온디바이스 Gemma의 응답을 받는 중심 화면입니다. 하단에는 탭 내비게이션이 있으며, 상담 중에도 갤러리, 건강, 패턴, 설정 화면으로 이동할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/대화.jpg" width="320" alt="상담 대화 화면">

상담 응답은 다음 흐름을 따릅니다.

- 사용자의 감정과 상황을 먼저 반영합니다.
- 바로 해결책을 단정하지 않고 선택 가능한 관점을 제시합니다.
- 공부, 생활, 관계, 건강 등 문제를 나누어 사용자가 이야기할 방향을 고를 수 있게 합니다.
- 위험 신호가 있으면 현재 안전 여부를 먼저 확인합니다.
- 의료, 법률, 재정 등 전문 영역은 전문 기관 상담을 권합니다.

### 2. 갤러리 분석

갤러리 탭은 사용자의 이미지에서 생활 맥락을 추출하는 화면입니다. 이미지 분석 진행률, 분석 문서, 다시 분석, 권한 줄이기, 썸네일 목록을 제공합니다.

<img src="./종설_wkfy/결과%20사진/갤러리1.jpg" width="320" alt="갤러리 분석 화면">

분석 문서는 상담에 바로 쓰이는 내부 요약입니다. 단순히 사진을 분류하는 것이 아니라, 상담에서 먼저 확인해야 할 주제와 질문 방향을 정리합니다.

<img src="./종설_wkfy/결과%20사진/갤러리2.jpg" width="320" alt="갤러리 분석 결과 화면">

갤러리 분석 결과에는 다음 정보가 포함됩니다.

- 상담 진행 전략
- 안전 및 폭력 보조 확인
- 현재 대화 단계
- 지금 물어볼 핵심 질문
- 이후 활용 가능한 회복 자원 후보

이미지 분석은 사용자의 상태를 확정하는 용도가 아닙니다. 사진만으로 자해, 폭력, 성폭력, 피해 여부를 단정하지 않고, 직접 발화나 위험 신호가 있을 때 안전 확인을 우선합니다.

### 3. 건강 데이터

건강 탭은 Health Connect 기반 생활 리듬 요약 화면입니다. 사용자는 권한을 연결하고 최신 데이터를 새로고침할 수 있으며, 주간/월간 단위로 활동 정보를 확인할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/헬스1.jpg" width="320" alt="건강 데이터 요약 화면">

하단으로 스크롤하면 수면 시간까지 포함한 상세 요약을 확인할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/헬스2.jpg" width="320" alt="건강 데이터 상세 화면">

건강 화면에서 다루는 정보는 다음과 같습니다.

- 걸음 수
- 총 소모 칼로리
- 활동 칼로리
- 이동 거리
- 평균 심박수
- 수면 시간

이 데이터는 상담 응답에서 "최근 수면이 줄었으니 우울하다"처럼 단정적으로 사용하지 않습니다. 대신 사용자가 피로, 긴장, 무기력, 생활 리듬 변화를 이야기할 때 조심스럽게 참고하는 보조 맥락으로 사용됩니다.

### 4. 생활 패턴

생활 패턴 탭은 통화 기록과 앱 사용 정보를 바탕으로 최근 생활 패턴을 요약합니다. 사회적 연결, 고립 신호, 디지털 과사용, 야간 리듬을 한눈에 보여주고, 통화 패턴을 지난 주와 비교합니다.

<img src="./종설_wkfy/결과%20사진/피노타입.jpg" width="320" alt="생활 패턴 화면">

주요 지표는 다음과 같습니다.

- 사회적 연결: 주간 통화량과 고유 연락처 수
- 고립 신호: 연속 무연락 기간
- 디지털 과사용: 일평균 앱 사용 시간
- 야간 리듬: 야간 시간대 사용량
- 통화 패턴: 이번 주와 지난 주 통화량 비교

생활 패턴은 상담 중 사용자의 생활 변화를 물어볼지 판단하는 보조 신호입니다. 고립, 과사용, 야간 리듬 같은 표현도 진단명이 아니라 앱 내부에서 쓰는 상담 보조 지표입니다.

### 5. 권한 관리

앱은 민감한 데이터를 다루기 때문에 권한 허용을 명확하게 분리합니다. 갤러리, 통화 기록, 앱 사용 정보는 각각 별도의 권한 흐름을 거칩니다.

<img src="./종설_wkfy/결과%20사진/이미지%20접근권한.jpg" width="320" alt="이미지 접근 권한 화면">

갤러리 권한은 이미지 분석 문서를 만들 때 사용됩니다. 사용자는 제한된 액세스 허용, 모두 허용, 허용 안 함 중 선택할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/통화%20접근권한.jpg" width="320" alt="통화 접근 권한 화면">

통화 권한은 생활 패턴 탭에서 사회적 연결과 고립 신호를 계산하기 위해 사용됩니다.

<img src="./종설_wkfy/결과%20사진/앱%20접근권한.jpg" width="320" alt="앱 사용 접근 권한 화면">

앱 사용 접근 권한은 디지털 과사용과 야간 리듬을 계산하기 위해 사용됩니다. 사용자가 권한을 주지 않으면 해당 지표는 "권한 없음" 상태로 표시됩니다.

### 6. 설정

설정 화면은 상담에 들어가는 맥락과 앱 표시 방식을 조정하는 관리 화면입니다.

<img src="./종설_wkfy/결과%20사진/설정0.jpg" width="320" alt="설정 화면 모델 및 응답 옵션">

대화 관리 영역에서는 프롬프트 확인, 중요 기억, 새 세션, 세션 목록, 내보내기, 불러오기를 사용할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/설정1.jpg" width="320" alt="설정 화면 대화 관리">

응답 옵션과 표시 영역에서는 상담 응답에 들어갈 데이터 범위와 화면 표시 방식을 조정합니다.

<img src="./종설_wkfy/결과%20사진/설정2.jpg" width="320" alt="설정 화면 응답 옵션 및 표시">

설정에서 조정할 수 있는 항목은 다음과 같습니다.

- 모델 파일 선택
- 대화 관리 펼치기
- 응답 맥락 접기
- 건강 데이터 제외
- 생활 패턴 제외
- Gallery 분석 제외
- 분석 기간 설정
- 사고/위험 신호 자동 처리
- 오디오 직접 분석 켜기
- 글자 크기 조정
- 테마 설정

즉, 사용자는 상담 응답에 어떤 데이터가 들어갈지 직접 조정할 수 있습니다. 특정 데이터가 부담스럽거나 불필요하면 설정에서 제외할 수 있습니다.

### 7. 중요 기억

중요 기억은 상담 품질을 일정하게 유지하기 위한 기본 원칙과, 사용자가 대화 중 저장해 달라고 한 개인 맥락을 보여주는 화면입니다.

<img src="./종설_wkfy/결과%20사진/중요기억1.jpg" width="320" alt="기본 중요 기억 화면">

기본 기억에는 상담 응답 원칙이 저장됩니다.

- 상담 응답은 진단이나 치료를 대신하지 않습니다.
- 감정 반영, 정리, 작은 다음 행동 선택을 돕습니다.
- 불안이나 긴장이 높을 때는 부담 낮은 안정화 방법을 선택지로 제안합니다.
- 자살, 자해, 타해, 학대, 폭력, 응급 위험 신호가 있으면 안전 확인을 우선합니다.

사용자가 대화 중 "이건 중요해", "기억해줘", "저장해줘"처럼 말하면 사용자 기억으로 저장됩니다.

<img src="./종설_wkfy/결과%20사진/중요기억2.jpg" width="320" alt="사용자 중요 기억 화면">

사용자 기억은 이후 상담에서 선호 활동이나 안정화 방법을 추천할 때 참고됩니다. 사용자는 저장된 기억을 삭제할 수 있습니다.

### 8. 프롬프트 확인

프롬프트 화면은 상담 보조자가 실제로 어떤 지침과 맥락을 참고하는지 확인하고 적용하는 화면입니다.

<img src="./종설_wkfy/결과%20사진/프롬포트1.jpg" width="320" alt="기본 상담 프롬프트 화면">

기본 상담 지침은 사용자의 상황을 단정하지 않고, 감정 반영과 안전 확인을 우선하도록 구성됩니다.

<img src="./종설_wkfy/결과%20사진/프롬포트2.jpg" width="320" alt="건강 및 생활 패턴 프롬프트 화면">

건강 및 생활 패턴 프롬프트는 수면, 활동, 통화, 앱 사용 패턴을 상담에 참고할 수 있는 형태로 요약합니다.

<img src="./종설_wkfy/결과%20사진/프롬포트3.jpg" width="320" alt="갤러리 분석 프롬프트 화면">

갤러리 분석 프롬프트는 사진 기반 상담 전략을 담습니다. 단, 이미지 분석만으로 위험을 확정하지 않고 현재 안전 여부를 확인하는 질문을 우선합니다.

## 상담 응답 설계

Counseling의 응답은 한 번에 답을 주는 방식보다 대화를 이어가기 쉬운 구조를 지향합니다.

1. 사용자의 감정과 의도를 짧게 반영합니다.
2. 상황을 판단하거나 진단하지 않습니다.
3. 필요한 경우 한 번에 하나의 명확한 질문을 합니다.
4. 여러 해결책을 동시에 강요하지 않고 선택지를 줄입니다.
5. 사용자가 바로 실행할 수 있는 작은 행동을 제안합니다.
6. 위험 신호가 있으면 안전 확인과 긴급 지원 안내를 우선합니다.
