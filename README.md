# 온디바이스 상담 보조 앱

사용자의 갤러리 이미지, 건강 데이터, 생활 패턴을 바탕으로 현재 상태를 정리하고 온디바이스 Gemma가 상담 흐름을 보조하는 모바일 앱입니다.  
진단이나 치료를 대신하지 않고, 사용자가 자신의 상황을 안전하게 돌아보고 다음 행동을 선택할 수 있도록 돕는 것을 목표로 합니다.

## 핵심 사용 흐름

1. 사용자는 상담 탭에서 현재 고민을 입력합니다.
2. 앱은 갤러리, 건강, 생활 패턴 데이터를 보조 맥락으로 정리합니다.
3. 온디바이스 Gemma가 사용자의 말에 바로 조언하기보다 먼저 감정, 상황, 안전 여부를 확인합니다.
4. 필요한 경우 갤러리 분석 결과나 건강/패턴 요약을 상담 프롬프트에 반영합니다.
5. 사용자는 상담을 이어가며 자신의 상태를 정리하고, 필요한 다음 행동을 선택합니다.

## 화면 구성

### 1. 상담

상담 화면은 앱의 중심 흐름입니다. 사용자는 하단 입력창에 고민을 작성하고, 챗봇은 사용자의 질문을 바로 해결하려 하기보다 상황을 정리할 수 있는 관점을 제시합니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_135529794.jpg" width="320" alt="상담 화면">

상담 응답은 다음 원칙을 따릅니다.

- 사용자의 감정과 상황을 먼저 반영합니다.
- 단정적인 진단이나 치료 표현을 피합니다.
- 안전, 폭력, 자해 등 위험 신호가 보이면 우선 확인 질문을 합니다.
- 복잡한 문제는 한 번에 해결하려 하지 않고 선택 가능한 관점으로 나눕니다.

### 2. 갤러리 분석

갤러리 탭에서는 휴대폰 이미지 분석 상태와 분석 문서를 확인합니다. 분석된 사진은 운동, 식사, 공부, 외출, 사람 관계 등 생활 맥락을 파악하는 보조 자료로 사용됩니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_140301529.jpg" width="320" alt="갤러리 분석 화면">

주요 기능은 다음과 같습니다.

- 갤러리 이미지 분석 진행률 표시
- 분석 결과 문서 확인
- 이미지 재분석
- 갤러리 접근 권한 줄이기
- 분석된 이미지 썸네일 목록 확인

분석 문서는 상담에 직접 사용되는 전략 자료입니다. 예를 들어 안전 확인, 대화 단계, 현재 집중해야 할 질문 등이 정리됩니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_135529794_01.jpg" width="320" alt="갤러리 분석 결과 문서">

### 3. 건강 데이터

건강 탭에서는 Health Connect 권한을 연결한 뒤 걸음 수, 칼로리, 이동 거리, 평균 심박수, 수면 시간 등을 주간 또는 월간 단위로 확인합니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_135529794_02.jpg" width="320" alt="건강 데이터 요약 화면">

건강 데이터는 사용자의 상태를 단정하는 용도가 아니라 상담에 참고할 생활 리듬 자료로 사용됩니다.

- 걷기와 이동 거리로 활동량 흐름 확인
- 평균 심박수로 최근 컨디션 참고
- 수면 시간으로 피로와 생활 리듬 참고
- 새로고침으로 최신 데이터 반영

### 4. 생활 패턴

패턴 탭은 통화 기록과 앱 사용 통계를 기반으로 사회적 연결, 고립 신호, 디지털 과사용, 야간 리듬 등을 요약합니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_135529794_04.jpg" width="320" alt="생활 패턴 화면">

화면은 핵심 지표 요약과 세부 패턴으로 나뉩니다.

- 사회적 연결: 최근 통화량과 연락처 수 변화
- 고립 신호: 연속 무연락 기간
- 디지털 과사용: 일평균 스크린타임
- 야간 리듬: 야간 사용 시간

이 정보는 상담에서 사용자를 판단하기 위한 기준이 아니라, 최근 생활 변화가 있는지 조심스럽게 확인하기 위한 보조 신호입니다.

### 5. 프롬프트 설정

설정 화면에서는 상담 보조자의 기본 지침과 각 데이터 영역별 자동 주입 맥락을 확인하고 수정할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_142433257.jpg" width="320" alt="기본 상담 프롬프트 설정">

프롬프트는 다음 영역으로 분리됩니다.

- 기본 상담 지침
- 건강 데이터 요약
- 생활 패턴 요약
- 갤러리 이미지 분석

사용자는 각 프롬프트를 확인한 뒤 적용 버튼으로 상담 흐름에 반영할 수 있습니다.

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_142433257_01.jpg" width="320" alt="건강 및 생활 패턴 프롬프트">

<img src="./종설_wkfy/결과%20사진/KakaoTalk_20260607_142433257_02.jpg" width="320" alt="갤러리 분석 프롬프트">

## 전체 사용자 시나리오

1. 사용자가 앱을 실행하고 상담 탭에서 고민을 입력합니다.
2. 갤러리 탭에서 최근 이미지 분석 결과를 확인하거나 다시 분석합니다.
3. 건강 탭에서 Health Connect 권한을 연결하고 최근 활동/수면 데이터를 갱신합니다.
4. 패턴 탭에서 통화, 앱 사용, 야간 리듬 등 생활 패턴 지표를 확인합니다.
5. 설정에서 상담 프롬프트와 데이터별 자동 주입 맥락을 조정합니다.
6. 다시 상담 탭으로 돌아와 보조 맥락이 반영된 대화를 이어갑니다.

## 설계 방향

- **온디바이스 우선**: 민감한 상담 맥락을 외부 서버보다 기기 내부에서 처리하는 방향을 지향합니다.
- **보조 자료 중심**: 갤러리, 건강, 생활 패턴 데이터는 판단 근거가 아니라 대화를 돕는 참고 자료입니다.
- **안전 우선 응답**: 위험 신호가 감지되면 일반 조언보다 안전 확인을 우선합니다.
- **사용자 통제**: 권한 연결, 권한 줄이기, 프롬프트 수정, 재분석 기능을 통해 사용자가 데이터 활용 범위를 조절할 수 있습니다.
